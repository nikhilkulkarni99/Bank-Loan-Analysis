🏦 Bank Loan Analysis Dashboard (SQL + Power BI) ----

📌 Project Overview ----
This project presents an end-to-end data analysis solution for bank loan data using MySQL and Power BI. The goal is to analyze loan distribution, risk patterns, and customer behavior to derive actionable business insights.

🛠️ Tools & Technologies ----
SQL (MySQL Workbench)
Power BI
Excel (initial data handling)

🔄 Workflow ----
Data Cleaning (SQL)
Converted date formats
Handled missing values
Created analytical views
Data Modeling (Power BI)
Built calculated columns (Risk Category, Income Group)
Created DAX measures for KPIs

Dashboard Development ----
Page 1: Executive Summary
Page 2: Risk & Customer Analysis
Page 3: Insights & Recommendations

📊 Key Metrics ----
Total Loan Amount
Total Payments
Average Interest Rate
Default Rate
Average DTI

🔍 Key Insights ----
Loan demand is concentrated in low and lower-middle income groups.
High-risk borrowers are charged higher interest rates.
Higher DTI correlates with increased default probability.
Debt consolidation dominates loan purposes.
Loan distribution is concentrated in key geographic regions.
Lower credit grades carry higher risk.

💡 Business Recommendations ----
Strengthen credit checks for high DTI borrowers.
Focus on mid-income segments for growth.
Improve data quality for geographical fields.
Monitor high-risk segments closely.
